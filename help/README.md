# Frontend

VueJS app that serves as a client.

## Dev requirements
* [Yarn](https://classic.yarnpkg.com/en/docs/install)

---
## Project setup

```
yarn install
```

---
## Development

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
